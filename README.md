#  FastDesk — Front-end

FastDesk é um **ERM (Enterprise Resource Management)** em desenvolvimento, criado com o objetivo de aplicar conceitos práticos de engenharia de software, com foco especial em estrutura de dados, algoritmos eficientes e boas práticas de desenvolvimento front-end.

Este repositório contém o front-end da aplicação, desenvolvido em React, e faz parte de um projeto colaborativo entre amigos, voltado tanto para aprendizado profundo quanto para construção de uma base sólida e escalável.

---

##  Objetivos do Projeto

- Desenvolver uma aplicação front-end moderna e bem estruturada
- Aplicar conceitos de Estrutura de Dados no contexto real de uma aplicação
- Implementar algoritmos performáticos, analisando:
  - Complexidade de tempo (Big-O)
  - Complexidade de espaço
- Praticar organização de código, componentização e reutilização
- Documentar decisões técnicas e pontos importantes diretamente no código
- Simular um ambiente próximo ao desenvolvimento profissional em equipe

---

Este projeto prioriza:

- Uso consciente de estruturas como:
  - Arrays
  - Hash Maps
  - Pilhas
  - Filas
  - Listas
- Implementação de algoritmos com boa complexidade
- Análise de trade-offs entre legibilidade e performance
- Comentários explicativos em trechos críticos do código
- Clareza no fluxo de dados e no estado da aplicação

---

##  Tecnologias Utilizadas

- **React** — biblioteca principal para construção da interface
- **JavaScript (ES6+)**
- **Vite** — ambiente de desenvolvimento rápido e moderno
- **Git & GitHub** — versionamento e colaboração
